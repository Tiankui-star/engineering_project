# 侧边栏样式优化说明

## 📋 优化背景
由于数据大屏页面占据较大空间，原侧边栏显得过小，导致页面比例失衡。本次优化增大了侧边栏的整体尺寸，使其与数据大屏更加协调。

## 🔧 主要调整内容

### 1. 侧边栏宽度调整
**文件**: `src/assets/styles/variables.scss`

```scss
// 修改前
$base-sidebar-width: 200px;

// 修改后
$base-sidebar-width: 240px;  // 增加40px，提升20%
```

**效果**: 侧边栏从 200px 增加到 240px，菜单文字显示更舒适。

---

### 2. 菜单项尺寸调整
**文件**: `src/assets/styles/sidebar.scss`

#### 2.1 菜单项高度和字体
```scss
// 修改前
.el-menu-item, .el-submenu__title {
  height: 44px !important;
  line-height: 44px !important;
}

// 修改后
.el-menu-item, .el-submenu__title {
  height: 50px !important;      // 增加6px
  line-height: 50px !important;  // 增加6px
  font-size: 15px !important;    // 新增字体大小
}
```

#### 2.2 图标尺寸
```scss
// 修改前
.svg-icon {
  margin-right: 10px !important;
}

// 修改后
.svg-icon {
  margin-right: 12px !important;  // 增加间距
  width: 18px !important;          // 新增图标宽度
  height: 18px !important;         // 新增图标高度
}
```

#### 2.3 子菜单字体
```scss
// 新增：子菜单字体稍小，保持层级关系
& .nest-menu .el-submenu>.el-submenu__title,
& .el-submenu .el-menu-item {
  font-size: 14px !important;
}
```

---

### 3. Logo区域调整
**文件**: `src/layout/components/Sidebar/Logo.vue`

#### 3.1 Logo容器高度
```scss
// 修改前
height: 50px;
line-height: 50px;

// 修改后
height: 56px;      // 增加6px
line-height: 56px;  // 增加6px
```

#### 3.2 Logo图标尺寸
```scss
// 修改前
width: 35px;
height: 35px;

// 修改后
width: 40px;   // 增加5px
height: 40px;  // 增加5px
```

#### 3.3 标题字体
```scss
// 修改前
font-size: 16px;
line-height: 50px;

// 修改后
font-size: 17px;   // 增大1px
line-height: 56px;  // 与容器高度一致
```

#### 3.4 折叠状态调整
```scss
// 修改前
&.collapse {
  width: 54px;
  height: 50px;
  
  .sidebar-logo {
    width: 30px;
    height: 30px;
  }
}

// 修改后
&.collapse {
  width: 64px;      // 增加10px
  height: 56px;     // 增加6px
  
  .sidebar-logo {
    width: 34px;    // 增加4px
    height: 34px;   // 增加4px
  }
}
```

---

### 4. 折叠侧边栏宽度调整
**文件**: `src/assets/styles/sidebar.scss`

```scss
// 修改前
.hideSidebar {
  .sidebar-container {
    width: 54px !important;
  }
  .main-container {
    margin-left: 54px;
  }
}

// 修改后
.hideSidebar {
  .sidebar-container {
    width: 64px !important;  // 增加10px
  }
  .main-container {
    margin-left: 64px;       // 增加10px
  }
}
```

---

### 5. 滚动区域高度调整
**文件**: `src/assets/styles/sidebar.scss`

```scss
// 修改前（适配旧的Logo高度50px）
&.has-logo {
  .el-scrollbar {
    height: calc(100% - 50px);
  }
}

// 修改后（适配新的Logo高度56px）
&.has-logo {
  .el-scrollbar {
    height: calc(100% - 56px);
  }
}
```

## 📊 优化前后对比

| 项目 | 优化前 | 优化后 | 增幅 |
|------|--------|--------|------|
| 侧边栏宽度 | 200px | 240px | +20% |
| 菜单项高度 | 44px | 50px | +13.6% |
| 菜单项字体 | 默认 | 15px | 新增 |
| 图标尺寸 | 默认 | 18×18px | 新增 |
| Logo高度 | 50px | 56px | +12% |
| Logo图标 | 35×35px | 40×40px | +14.3% |
| 标题字体 | 16px | 17px | +6.25% |
| 折叠宽度 | 54px | 64px | +18.5% |

---

## 🎯 优化效果

1. **视觉平衡**: 侧边栏与数据大屏的比例更加协调
2. **可读性提升**: 更大的字体和图标，阅读更舒适
3. **点击区域增大**: 菜单项高度增加，点击更精准
4. **层级清晰**: 一级菜单15px，子菜单14px，保持视觉层次
5. **响应式兼容**: 折叠状态同步调整，保持一致性

---

## 💡 使用说明

### 生效方式
修改完成后，需要重新编译前端项目：

```bash
cd ruoyi-ui
npm run dev    # 开发环境
npm run build  # 生产环境
```

### 回滚方法
如需恢复原始样式，只需将上述修改的数值改回即可。关键文件：
- `src/assets/styles/variables.scss`
- `src/assets/styles/sidebar.scss`
- `src/layout/components/Sidebar/Logo.vue`

---

## 📝 注意事项

1. **浏览器缓存**: 修改后建议清除浏览器缓存或强制刷新（Ctrl+F5）
2. **主题兼容**: 优化同时适配了 `theme-dark` 和 `theme-light` 两种主题
3. **移动端**: 移动端响应式布局不受影响
4. **自定义配置**: 如需进一步调整，可修改 `variables.scss` 中的 `$base-sidebar-width` 变量
